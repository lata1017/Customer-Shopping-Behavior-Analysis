# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview
This project analyzes customer shopping behavior using Python, SQL, and Power BI to extract meaningful insights about purchasing patterns, customer segments, and revenue trends. The goal is to help businesses understand customer preferences, spending habits, product performance, and the impact of subscriptions on revenue.

## 📊 Dashboard Highlights
- Total Customers: 627  
- Average Purchase Amount: $60.73  
- Average Review Rating: 3.77  
- Subscription Distribution: No (75.6%), Yes (24.4%)  
- Top Revenue Category: Clothing  
- Customer Segmentation by Age Group  

## 🧰 Tech Stack
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- SQL  
- Power BI  

## 📁 Project Structure
├── Customer_Shopping_Behavior_Analysis.ipynb  
├── Customer-sql.sql

├── customer_behavior_dashboard.pbix  
├── customer_behavior_dashboard.pdf  
└── README.md  

## 🔍 Key Analysis Performed
Customer Analysis: Gender-wise revenue distribution, age group segmentation, customer classification (New, Returning, Loyal).  
Revenue Analysis: Revenue by category, revenue by age group, subscription vs non-subscription revenue.  
Product Insights: Top-selling products, highest-rated products, discount impact on purchases.  

## 🧠 SQL Insights
Revenue by gender, top 5 products by rating, discount usage analysis, customer segmentation using CASE statements, ranking products using window functions.  

## 📈 Power BI Features
Interactive filters: Gender, Category, Shipping Type, Subscription Status.  
Visualizations: KPI Cards, Bar Charts, Donut Chart, Revenue Analysis Charts.  

## 🚀 How to Run the Project
1. Python Analysis: Open the notebook and install required libraries using `pip install pandas numpy matplotlib seaborn`.  
2. SQL Queries: Run queries in any SQL tool (MySQL/PostgreSQL/SQL Server).  
3. Power BI Dashboard: Open the `.pbix` file in Power BI Desktop and refresh the data.  

## 💡 Key Insights
Clothing category generates the highest revenue. Majority customers are non-subscribers. Young adults contribute the most revenue. Discounts influence purchase behavior. Loyal customers contribute significantly to total revenue.
